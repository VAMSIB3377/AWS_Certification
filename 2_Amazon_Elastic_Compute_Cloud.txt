2_Amazon_Elastic_Compute_Cloud ( EC2 )


Instance - A Server Running Our Application is called an Instance

--> EC2 is Highly Flexible, Cost-Effective and Quick When compared to you running own servers on premises.

--> In EC2 you only pay for what you use, in EC2 you'll only pay for the instances which are running, not stopped, not terminated.

--> EC2 runs on top of physical host machines managad by AWS using virtualization Technology.

--> EC2 instances are also resizable, which is vertical-scaling.
